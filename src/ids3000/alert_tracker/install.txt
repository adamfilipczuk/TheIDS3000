Python Watchdog Installation
============================

Requirements:
-------------
- Python 3.6 or higher
- pip package manager

Install watchdog library:
------------------------
pip install watchdog

(Optional) If using a virtual environment:
-------------------------------------------
python -m venv venv
# Activate venv:
# Linux/macOS: source venv/bin/activate
# Windows:    venv\Scripts\activate
pip install watchdog
